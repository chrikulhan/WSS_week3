Pick five Minnesota state parks at random
and print their info for your brochure:

This program was built to extract information from current Minnesota State Park websites. It allows the programmer to automatically organize the information with correct headings and consistent formatting. The aggregated information is automatically saved to a word document and could be made into a brochure. The code below is a snippet of the total code. 


import random
import requests
from pprint import pprint
import docx

park_list_url = 'https://mn-state-parks.herokuapp.com/api/list'
park_list = requests.get(park_list_url).json()  # gets that data
five_random_parks = random.sample(park_list, 5)  # chooses 5 parks

selected_parks_urls_list = []

for park in five_random_parks:  # this will grab 5 parks from all the parks and save them to the above list
    park_id = park['park_id']
    park_url = 'https://mn-state-parks.herokuapp.com/api/' + park_id
    selected_parks_urls_list.append(park_url)
